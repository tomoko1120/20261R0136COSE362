# AI + Climate Resilience: Heavy Rainfall Prediction Using VAE-Based Data Augmentation

This public repository contains the final project code for COSE362 ML.

## Main Notebook
Please run:

implementation.ipynb

This notebook includes:
1. Dataset loading
2. Feature selection and preprocessing
3. HeavyRainTomorrow label creation
4. Train/test split and standardization
5. VAE training on heavy-rain samples
6. Synthetic data generation
7. Logistic Regression and Random Forest classification
8. Evaluation using Recall, Precision, and F1-score

## Dataset
dataset.csv is included in the repository.

## Additional Experiment Notebooks
The experiments/ folder contains additional ablation experiments, including activation-function testing and GMM-related experiments.

## How to Run
Open implementation.ipynb in Google Colab or Jupyter Notebook.
Run all cells from top to bottom.
